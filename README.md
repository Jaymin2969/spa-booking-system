# Natureland Spa Management System

A comprehensive wellness and spa booking management system built with React.

## Features
- **Interactive Calendar**: Unified view of therapist schedules and bookings.
- **Booking Management**: Create, edit, and track service appointments.
- **Therapist Portal**: Manage professional profiles and specialized services.
- **Outlet Management**: Handle multiple spa locations and room assignments.

## Getting Started

### Local Development
1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Built With
- React
- Axios
- Zustand (State Management)
- Lucide React (Icons)
